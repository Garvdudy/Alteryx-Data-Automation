# Alteryx Data Automation Portfolio

This repository contains enterprise-style data automation projects
focused on cleaning, transforming, validating, and summarizing banking, business
and client datasets using modern analytics tools.

The goal of this portfolio is to demonstrate hands-on experience in:

-   Data preparation and transformation
-   Workflow automation
-   Risk flag logic implementation
-   Aggregation and reporting automation
-   Process optimization in financial datasets
-   Reproducible ETL-style pipelines

------------------------------------------------------------------------

## Tools & Technologies Used

-   Alteryx Designer
-   SQL
-   Microsoft Excel
-   Data modeling principles
-   ETL automation concepts

------------------------------------------------------------------------

## Projects Included

### CIBC Client Transaction Data Cleaning & Risk Automation

A simulated Canadian banking dataset was cleaned and transformed using
Alteryx workflows to:

-   Standardize inconsistent formats
-   Handle null and invalid values
-   Implement automated risk flag logic
-   Generate client-level and regional summaries
-   Replace manual spreadsheet-based processes

Includes:

-   Raw dataset
-   Alteryx workflow file (.yxmd)
-   Cleaned dataset output
-   Risk summary reports
-   Full project documentation
